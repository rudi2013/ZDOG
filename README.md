ZDOG is a Cytoscape 3.0 app. It runs in Cytoscape 3.6.1 and 3.7.1.

## Short instruction

To install ZDOG, first install Cytoscape. Start Cytoscape and choose Apps - App manager. Search ZDOG and click Install.
